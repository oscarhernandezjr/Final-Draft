My To-Do List

In this project, we were able to build a to-do list maker. We used html to input our information, css to style it, and javascript to make it function.

Users can add or remove a task with it's easy to use format and color-coded display.

Number Guessing Game

This game allows users to guess numbers from 1-100. We can all admit the odds are very slim, therefore, we gave users ten tries as well as advising them if it was too high or too low.

Make A Pizza!

This was one of my first projects in my html educational journey. It let users to input their preferred pizza, allowing to customize it to their likings.

It then shows a confirmation with one's delivery informaton as well as their pizza of choice.
